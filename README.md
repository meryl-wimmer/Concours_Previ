# Concours Prévi

These scripts download MF and DWD models data and print values of T2m, RR and visibilty at Blagnac location, in order to participate at Concours Prévi.

For MF models participation : 
Change date (line 3) in Concours_previ_MF.sh
and run this script : ./Concours_previ_MF.sh

For DWD models participation : 
Change date (line 3) in Concours_previ_DWD.sh
and run this script : ./Concours_previ_DWD.sh

Do not forget to empty folders named data_MF and data_DWD when finished.
